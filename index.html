0.1 故事的起點：熱傳導方程式 (1807)

1807 年，法國數學家暨物理學家 Jean-Baptiste Joseph Fourier（1768–1830）向法國科學院（Institut de France）提交了一篇關於熱傳導的論文。他試圖解決一個看似簡單的問題：

核心問題

「一塊金屬板，已知初始溫度分布，溫度隨時間如何變化？」

他推導出的熱傳導方程式 (Heat Equation)：

$$\frac{\partial u}{\partial t} = \alpha \frac{\partial^2 u}{\partial x^2}$$

$u(x,t)$：位置 $x$、時間 $t$ 的溫度；$\alpha$：熱擴散係數 (Thermal Diffusivity)

為了求解這個偏微分方程式（PDE），Fourier 提出了一個石破天驚的主張：

Fourier 的核心主張

「任意函數——包括不連續的函數——都可以展開為正弦與餘弦函數的無窮級數。」

$$f(x) = \frac{a_0}{2} + \sum_{n=1}^{\infty}\left[a_n \cos(nx) + b_n \sin(nx)\right]$$

這個主張在當時極具爭議性。數學家們無法相信像鋸齒波、方波這樣有「尖角」或「跳躍」的函數，竟然可以用無限光滑的正弦波來表示。

0.2 Lagrange 的反對與數學爭論

Fourier 的論文由三位當時最頂尖的數學家審查：Lagrange、Laplace 和 Legendre。其中 Lagrange 強烈反對：

Lagrange 的論點

「三角級數（正弦/餘弦的和）在每一點都是連續且可微的，怎麼可能表示一個有跳躍或尖角的函數？」

Lagrange 的直覺是對的 — 有限項的三角級數確實是光滑的。但他忽略了一個關鍵點：無窮級數的行為可以和有限項截然不同。無窮多個光滑函數的和，可以逼近不光滑的函數。

由於 Lagrange 的反對，Fourier 的論文直到 1822 年才以專著《熱的解析理論》（Théorie analytique de la chaleur）的形式正式出版。

歷史啟示：這場爭論催生了現代數學分析中「收斂性」的嚴格研究。數學家們開始思考：三角級數到底在什麼條件下會收斂到原函數？收斂的方式（逐點、均勻、L² 意義下）又有何不同？

0.3 Dirichlet 收斂條件 (1829)

1829 年，德國數學家 Peter Gustav Lejeune Dirichlet 給出了傅立葉級數收斂的第一組嚴格充分條件 (Sufficient Conditions)：

Dirichlet 條件 (Dirichlet Conditions)

函數 $f(x)$ 的傅立葉級數在區間 $[a, b]$ 上收斂，若滿足：

  1. $f(x)$ 在 $[a,b]$ 上絕對可積：$\int_a^b |f(x)|\,dx < \infty$
  2. $f(x)$ 在任何有限區間內只有有限個不連續點
  3. $f(x)$ 在任何有限區間內只有有限個極值點（有界變差 Bounded Variation）

在不連續點 $x_0$ 處，級數收斂到左右極限的平均值：$\frac{1}{2}\left[f(x_0^+) + f(x_0^-)\right]$

工程意義：幾乎所有物理上可實現的訊號都滿足 Dirichlet 條件。方波、三角波、鋸齒波全部滿足。只有純粹人為構造的數學病態函數才會違反。因此在工程應用中，你可以放心地使用傅立葉分析。

0.4 FFT 革命：從 O(N²) 到 O(N log N) (1965)

傅立葉分析的理論在 19 世紀就已成熟，但直到 20 世紀中期，它在實際工程中的應用受限於計算能力。對一個 N 點的離散訊號做 DFT 需要 $O(N^2)$ 次複數乘法：

1965 年，James W. Cooley（IBM）和 John W. Tukey（普林斯頓大學）發表了劃時代的論文，提出了 Cooley-Tukey FFT 演算法。

冷戰背景

FFT 的開發動機來自核武禁試條約的監測需求。美國需要分析地震波的頻譜來區分地下核試爆和自然地震，這需要高效的頻譜分析演算法。Tukey 在一次國防顧問會議上提出了分治法的構想，Cooley 隨後在 IBM 將其實作為程式。

有趣的歷史：事後發現，高斯 (Gauss) 早在約 1805 年就獨立發現了本質相同的演算法，用於計算小行星軌道的三角多項式插值。但高斯的手稿以拉丁文書寫且未廣泛傳播，直到 1960 年代才被重新發現。

現代 — FFT 無所不在

4G/5G OFDM、MP3/AAC 音訊壓縮、MRI 成像、語音辨識、AI — 每支智慧型手機每秒執行數百萬次 FFT

0.6 從熱方程到現代工程的連結

Fourier 當初為了解一個物理問題而發明的數學工具，如今已遠遠超出熱傳導的範疇。其核心洞察是：

「複雜的訊號可以分解為簡單的頻率成分；在頻率域中分析往往比在時間域更加容易。」

分析問題

訊號包含哪些頻率？各成分有多強？→ 傅立葉正轉換

例：電力系統 THD 分析、振動故障診斷、音訊頻譜分析

設計問題

如何組合不同頻率的成分來產生所需的訊號？→ 傅立葉反轉換

例：OFDM 訊號產生、合成音效、MRI 影像重建

處理問題

如何選擇性地保留或移除某些頻率成分？→ 頻域濾波

例：雜訊濾除、等化器、帶通濾波器設計

效率問題

如何高效計算卷積、相關等運算？→ FFT 加速

例：快速卷積、匹配濾波、大規模資料處理

1. 基礎概念

時間域 vs 頻率域 — 傅立葉分析的核心思想

學習目標

  • 區分時間域與頻率域表示法，理解兩者的互補關係
  • 說明傅立葉分析在 EE 各領域（通訊、電力、音訊、振動）的不可替代性
  • 從合成訊號中辨識出各頻率成分

1.1 時間域 (Time Domain) vs 頻率域 (Frequency Domain)

在工程分析中，觀察訊號有兩個基本視角：

  • 時間域：振幅隨時間的變化。示波器（Oscilloscope）顯示的波形。
  • 頻率域：訊號包含哪些頻率成分及其強度。頻譜分析儀（Spectrum Analyzer）顯示的資訊。

傅立葉分析的核心思想

任何滿足 Dirichlet 條件的週期訊號，都可以唯一分解為不同頻率正弦波之疊加。

$$x(t) = \sum_{k=-\infty}^{\infty} c_k \, e^{j k \omega_0 t}$$

1.2 為什麼 EE 工程師必須學傅立葉分析？

通訊系統 (Communication)

4G LTE / 5G NR 使用 OFDM（正交分頻多工），其核心運算就是 IFFT（發射端）和 FFT（接收端）。每個子載波（Subcarrier）就是一個傅立葉基底函數。

電力品質 (Power Quality)

IEEE 519 標準規範電力系統的總諧波失真 (THD)限值。用 FFT 分析電流/電壓的諧波含量是電力工程師的日常工作。

$\text{THD} = \frac{\sqrt{\sum_{n=2}^{N} V_n^2}}{V_1} \times 100\%$

醫學影像 (Medical Imaging)

MRI 直接在頻率域（k-space）擷取資料，再透過 2D IFFT 重建影像。理解傅立葉轉換 = 理解 MRI 的物理原理。

振動與故障診斷 (Vibration)

旋轉機械的軸承故障特徵頻率 (BPFO/BPFI)可透過包絡譜（Envelope Spectrum）分析偵測。這是預測性維護（Predictive Maintenance）的核心技術。

互動圖表：時間域 ↔ 頻率域

調整三個正弦波的頻率與振幅，觀察合成訊號在時間域和頻率域的變化。

2. 傅立葉級數 (Fourier Series)

週期訊號的頻率分解

學習目標

  • 寫出三角形式與複數指數形式的傅立葉級數公式
  • 計算方波、三角波、鋸齒波的傅立葉係數
  • 解釋 Gibbs 現象及其在 DAC、數位通訊中的工程影響

2.1 三角形式 (Trigonometric Form)

對於週期 $T$、基頻 $\omega_0 = 2\pi/T$ 的週期訊號 $x(t)$：

$$x(t) = \frac{a_0}{2} + \sum_{n=1}^{\infty}\left[a_n\cos(n\omega_0 t) + b_n\sin(n\omega_0 t)\right]$$

$a_n = \frac{2}{T}\int_0^T x(t)\cos(n\omega_0 t)\,dt$，$b_n = \frac{2}{T}\int_0^T x(t)\sin(n\omega_0 t)\,dt$

2.2 複數指數形式 (Complex Exponential Form)

$$x(t) = \sum_{n=-\infty}^{\infty} c_n\,e^{jn\omega_0 t}, \quad c_n = \frac{1}{T}\int_0^T x(t)\,e^{-jn\omega_0 t}\,dt$$

關係：$c_n = \frac{1}{2}(a_n - jb_n)$，$c_{-n} = c_n^*$（實數訊號的共軛對稱性）。

2.3 經典波形的傅立葉級數

方波 (Square Wave)

$$x(t) = \frac{4}{\pi}\sum_{n=1,3,5,\ldots}\frac{1}{n}\sin(n\omega_0 t)$$

只有奇數次諧波，振幅以 $1/n$ 衰減。半波對稱性 (Half-Wave Symmetry) 的結果。

三角波 (Triangle Wave)

$$x(t) = \frac{8}{\pi^2}\sum_{n=1,3,5,\ldots}\frac{(-1)^{(n-1)/2}}{n^2}\sin(n\omega_0 t)$$

振幅以 $1/n^2$ 衰減（比方波更快收斂，因為三角波是連續的）。

鋸齒波 (Sawtooth Wave)

$$x(t) = \frac{2}{\pi}\sum_{n=1}^{\infty}\frac{(-1)^{n+1}}{n}\sin(n\omega_0 t)$$

包含所有次諧波，振幅以 $1/n$ 衰減。

互動實驗：諧波合成與 Gibbs 現象

選擇波形並調整諧波數，觀察不連續點的 ~9% 過衝（Gibbs Phenomenon）。

2.4 Gibbs 現象的工程影響

有限項傅立葉級數在不連續點附近出現約 9% 的過衝，項數增加也不會消失（只是過衝區域變窄）。

DAC 重建濾波器

數位類比轉換器（DAC）輸出的階梯波經由重建濾波器（Reconstruction Filter）後，在快速跳變處會出現 ringing，這正是 Gibbs 現象的物理體現。高品質音訊 DAC 使用 oversampling + digital filter 來減緩此效應。

數位通訊中的符號間干擾 (ISI)

頻寬受限的通道會截斷訊號的高頻諧波，導致脈衝波形在符號邊界產生 Gibbs-like ringing，造成符號間干擾。Raised-cosine 脈衝整形就是為了消除此問題。

2.5 收斂條件補充

回顧第 0 章介紹的 Dirichlet 條件。傅立葉級數的收斂方式其實有多種層次：

收斂類型定義工程意義
逐點收斂
(Pointwise)		在每一點 $x$，$S_N(x) \to f(x)$大部分情況夠用，但在不連續點收斂到左右極限的平均
均勻收斂
(Uniform)		$\sup_x |S_N(x)-f(x)| \to 0$只有連續函數才能均勻收斂；方波做不到（Gibbs 現象）
均方收斂
(Mean-Square / $L^2$)		$\int|S_N(x)-f(x)|^2\,dx \to 0$能量意義下的收斂，Parseval 定理保證。工程中最常用。

實務指引：在工程應用中，你幾乎永遠只需要關心 $L^2$ 收斂 — 它等同於「能量守恆」。只要 Parseval 定理成立（時域能量 = 頻域能量），你的頻譜分析就是正確的。

3. 連續時間傅立葉轉換 (CTFT)

Continuous-Time Fourier Transform — 非週期訊號的頻率分析

學習目標

  • 寫出 CTFT 正反轉換公式，理解存在條件
  • 運用時移、卷積、Parseval 等性質於工程分析
  • 辨識 rect-sinc、Gaussian 等常見轉換對

3.1 定義

正轉換（Analysis）：

$$X(j\omega) = \int_{-\infty}^{\infty} x(t)\, e^{-j\omega t}\, dt$$

反轉換（Synthesis）：

$$x(t) = \frac{1}{2\pi}\int_{-\infty}^{\infty} X(j\omega)\, e^{j\omega t}\, d\omega$$

存在條件：嚴格來說，CTFT 存在需要 $x(t)$ 絕對可積：$\int_{-\infty}^{\infty}|x(t)|\,dt < \infty$。但工程中常遇到不滿足此條件的訊號（如 $\sin(\omega_0 t)$），此時需要引入分佈理論 (Distribution Theory) 和 Dirac delta 函數來定義廣義傅立葉轉換。

3.2 重要性質與工程解讀

性質數學表述工程意義
線性$ax_1+bx_2 \leftrightarrow aX_1+bX_2$多個訊號疊加，頻譜也直接疊加
時移$x(t-t_0) \leftrightarrow e^{-j\omega t_0}X(j\omega)$延遲 = 相位旋轉，幅度不變。這就是為什麼線性相位濾波器對應純延遲。
頻移$e^{j\omega_0 t}x(t) \leftrightarrow X(j(\omega-\omega_0))$調變 (Modulation) 的數學基礎。乘以載波 = 頻譜搬移。
尺度$x(at) \leftrightarrow \frac{1}{|a|}X(\frac{j\omega}{a})$時域壓縮 → 頻域展寬（不確定性原理）
卷積$x_1 * x_2 \leftrightarrow X_1 \cdot X_2$濾波 = 頻域乘法。這是所有濾波器理論的基石。
Parseval$\int|x|^2 dt = \frac{1}{2\pi}\int|X|^2 d\omega$能量守恆：時域和頻域計算能量結果相同

3.3 常見轉換對

$x(t)$$X(j\omega)$說明
$\text{rect}(t/\tau)$$\tau\,\text{sinc}(\omega\tau/2\pi)$矩形脈衝 ↔ sinc
$\delta(t)$$1$脈衝：所有頻率等量含有
$e^{-at}u(t)$$\frac{1}{a+j\omega}$單邊指數衰減（RC 電路）
$e^{-\alpha t^2}$$\sqrt{\frac{\pi}{\alpha}}\,e^{-\omega^2/(4\alpha)}$Gaussian ↔ Gaussian（最小不確定性）

  • 區分零填充帶來的「內插解析度」與「真正的頻率解析度」
  • 以 DFT 矩陣的觀點理解正交基底分解

4.1 DFT 定義

$$X[k] = \sum_{n=0}^{N-1} x[n]\, e^{-j2\pi kn/N}, \quad k = 0,1,\ldots,N-1$$ $$x[n] = \frac{1}{N}\sum_{k=0}^{N-1} X[k]\, e^{j2\pi kn/N}, \quad n = 0,1,\ldots,N-1$$

頻率解析度：$\Delta f = f_s / N$。N 點 DFT 將 $[0, f_s)$ 的頻率範圍均分為 N 個 bin。

4.2 DFT 矩陣觀點

DFT 可以表示為矩陣-向量乘法 $\mathbf{X} = \mathbf{W}_N \mathbf{x}$，其中：

$$[\mathbf{W}_N]_{kn} = e^{-j2\pi kn/N} = W_N^{kn}$$

$\mathbf{W}_N$ 是 $N \times N$ 的DFT 矩陣，其列向量互相正交。$\frac{1}{\sqrt{N}}\mathbf{W}_N$ 為酉矩陣（Unitary Matrix）。

幾何直覺：DFT 就是將訊號投影到 N 個正交的複指數基底 $\{e^{j2\pi kn/N}\}$ 上。每個 $X[k]$ 是訊號在第 $k$ 個基底上的「坐標值」。這與線性代數中的正交投影完全類比。

4.3 FFT：Cooley-Tukey Radix-2

$$X[k] = \underbrace{\sum_{r=0}^{N/2-1} x[2r]\,W_{N/2}^{rk}}_{\text{偶數項 DFT}} + W_N^k \underbrace{\sum_{r=0}^{N/2-1} x[2r+1]\,W_{N/2}^{rk}}_{\text{奇數項 DFT}}$$

一個 N 點 DFT 分解為兩個 N/2 點 DFT + N/2 次蝴蝶運算。遞迴下去，複雜度從 $O(N^2)$ 降至 $O(\frac{N}{2}\log_2 N)$。

4.4 零填充 (Zero-Padding)

  • 在訊號後補零，增加 DFT 長度 → 頻域取樣更密（內插）
  • 不會增加新的頻率資訊或提高真正的解析度
  • 讓 FFT 長度成為 2 的冪次 → 可使用 Radix-2 FFT

常見誤解：零填充不等於增加觀測時間。真正的頻率解析度 $\Delta f = 1/T_{\text{obs}}$ 取決於訊號的實際觀測長度。

互動：FFT 與零填充效果

訊號：50 Hz + 55 Hz，取樣率 500 Hz，觀測 0.1 秒。觀察零填充如何讓頻譜更平滑，但無法真正分開兩個頻率。

4.5 Radix-2 FFT JavaScript 實作

本網站使用的 FFT 函式完整原始碼（已內建）：

/** * Radix-2 Cooley-Tukey FFT * @param {number[]} real - 實部陣列（長度必須為 2 的冪次） * @param {number[]} imag - 虛部陣列 * @returns {{ real: number[], imag: number[] }} */ function fft(real, imag) { const N = real.length; if (N <= 1) return { real: [...real], imag: [...imag] }; const re = new Float64Array(N), im = new Float64Array(N); const bits = Math.round(Math.log2(N)); // 位元反轉排序 (Bit-Reversal Permutation) for (let i = 0; i < N; i++) { let rev = 0; for (let b = 0; b < bits; b++) rev = (rev << 1) | ((i >> b) & 1); re[rev] = real[i]; im[rev] = imag[i]; } // 蝴蝶運算 (Butterfly Operations) for (let size = 2; size <= N; size *= 2) { const half = size / 2, angle = -2 * Math.PI / size; for (let i = 0; i < N; i += size) { for (let j = 0; j < half; j++) { const wr = Math.cos(angle*j), wi = Math.sin(angle*j); const tr = wr*re[i+j+half] - wi*im[i+j+half]; const ti = wr*im[i+j+half] + wi*re[i+j+half]; re[i+j+half] = re[i+j] - tr; im[i+j+half] = im[i+j] - ti; re[i+j] += tr; im[i+j] += ti; } } } return { real: Array.from(re), imag: Array.from(im) }; }

5. 視窗函數 (Window Functions) 與頻譜洩漏 (Spectral Leakage)

頻譜分析中最關鍵的工程技巧

學習目標

  • 解釋頻譜洩漏的物理成因（有限長度截斷 = 頻域卷積）
  • 比較 5 種主流視窗的主瓣寬度、旁瓣衰減與等效雜訊頻寬
  • 根據「頻率解析度 vs 動態範圍」需求選擇合適的視窗

5.1 為什麼需要加窗？

截斷訊號 = 乘以矩形窗。在頻域中：

$$x_w[n] = x[n] \cdot w[n] \quad \longleftrightarrow \quad X_w(e^{j\omega}) = X(e^{j\omega}) * W(e^{j\omega})$$

矩形窗的頻譜是 sinc — 具有 -13 dB 的高旁瓣 → 嚴重洩漏

5.2 視窗函數公式

Rectangular

$$w[n] = 1$$

Hann (Hanning)

$$w[n] = 0.5\left(1-\cos\frac{2\pi n}{N-1}\right)$$

Hamming

$$w[n] = 0.54 - 0.46\cos\frac{2\pi n}{N-1}$$

Blackman

$$w[n] = 0.42 - 0.5\cos\frac{2\pi n}{N-1} + 0.08\cos\frac{4\pi n}{N-1}$$

Kaiser

$$w[n] = \frac{I_0\!\left(\beta\sqrt{1-\left(\frac{2n}{N-1}-1\right)^2}\right)}{I_0(\beta)}$$

6. 短時傅立葉轉換 (STFT) 與 Spectrogram

Short-Time Fourier Transform — 觀察頻率隨時間的變化

學習目標

  • 定義 STFT 並解釋不確定性原理對時頻解析度的限制
  • 選擇適當的視窗長度與重疊率
  • 解讀 Spectrogram 在音訊/振動分析中的典型模式

6.1 STFT 定義

$$\text{STFT}\{x[n]\}(m,\omega) = \sum_{n=-\infty}^{\infty} x[n]\,w[n-m]\,e^{-j\omega n}$$

$m$：時間索引；$w[n]$：視窗函數

不確定性原理：$\Delta t \cdot \Delta f \geq \frac{1}{4\pi}$。長視窗 → 頻率解析度高、時間解析度低；短視窗 → 反之。根據需求做取捨。

6.2 關鍵參數

參數影響典型值
視窗長度頻率解析度 $\Delta f = f_s/N_{\text{win}}$語音：20-40 ms；音樂：50-100 ms
重疊率時間軸取樣密度50%~75%
NFFT頻率軸密度（零填充）通常 ≥ 視窗長度，取 2 的冪次

6.3 工程應用場景

語音辨識

將 Spectrogram 轉為 Mel 刻度後作為深度學習模型的輸入特徵（Mel-Spectrogram / MFCC）。

旋轉機械振動監測

透過 Spectrogram 追蹤特徵頻率（如軸承缺陷頻率 BPFO）隨轉速或負載的變化趨勢。

7. 重要應用與定理

工程實務中的核心定理、產業應用場景與常見陷阱

學習目標

  • 運用卷積定理於濾波器設計與快速卷積
  • 陳述 Nyquist 定理並預測混疊發生的條件
  • 計算 THD 並判斷是否符合 IEEE 519 規範
  • 認識 OFDM 與軸承故障診斷中的傅立葉分析應用

7.1 卷積定理 (Convolution Theorem)

$$y(t) = x(t)*h(t) \;\longleftrightarrow\; Y(j\omega) = X(j\omega)\cdot H(j\omega)$$
  • 濾波 = 頻域乘法：低通濾波器 $H(j\omega)$ 在高頻為零 → 移除高頻噪聲。
  • 快速卷積：FFT → 逐點相乘 → IFFT，複雜度 $O(N\log N)$，比直接卷積 $O(N^2)$ 快。用於影像處理和長 FIR 濾波。

7.2 Nyquist-Shannon 取樣定理

$$f_s \geq 2f_{\max}$$

違反 → 混疊 (Aliasing)：高頻偽裝成低頻，無法事後修正。實務上需在取樣前加抗混疊濾波器 (Anti-Aliasing Filter)，且取樣率通常取 $>2.56\times f_{\max}$。

互動：Aliasing 混疊現象

7.3 功率譜密度 (PSD) 與 Welch 法

$$S_{xx}(f) = \lim_{T\to\infty}\frac{1}{T}|X_T(f)|^2$$

Welch 法：將訊號分段、加窗、FFT 後取平均。降低 PSD 估計的變異數（更穩定），代價是頻率解析度略降。是工程中估計 PSD 的標準方法。

7.4 總諧波失真 (THD) 分析

THD (Total Harmonic Distortion) 衡量一個訊號中非基頻成分所佔的比例，是電力品質、音訊設備規格中最重要的指標之一。

$$\text{THD} = \frac{\sqrt{V_2^2 + V_3^2 + V_4^2 + \cdots + V_N^2}}{V_1} \times 100\%$$

$V_1$：基頻振幅；$V_n$：第 $n$ 次諧波振幅

IEEE 519-2022 標準：電力系統中，一般用電設備的電流 THD 限值為 5%～8%（視系統短路比而定）。電壓 THD 限值通常為 5%。超標可能導致變壓器過熱、馬達振動、電容器損壞。

互動：THD 計算器

調整各次諧波的振幅（以基頻振幅百分比表示），即時計算 THD。

7.5 OFDM 通訊系統中的 FFT

正交分頻多工 (Orthogonal Frequency Division Multiplexing, OFDM) 是 4G LTE、5G NR、Wi-Fi 的核心調變技術。其本質就是：

發射端 — IFFT

將頻域上 N 個子載波的資料（QAM 符號）透過 IFFT 轉為時域訊號。每個子載波就是一個 DFT 基底函數 $e^{j2\pi kn/N}$。

接收端 — FFT

對接收到的時域訊號做 FFT，即可恢復各子載波上的資料。

循環前綴 (Cyclic Prefix, CP)：在每個 OFDM 符號前加上尾部的複製，可將線性卷積（通道效應）轉為循環卷積 → 保證 FFT 的正確性 → 消除符號間干擾 (Inter-Symbol Interference, ISI)。

5G NR 使用 4096 點 IFFT/FFT，子載波間距 15/30/60/120 kHz。你的手機每秒執行數百萬次 FFT。

7.6 旋轉機械振動分析與軸承故障診斷

在預測性維護 (Predictive Maintenance) 中，傅立葉分析是最核心的工具。軸承的四種基本缺陷各有特定的特徵頻率 (Characteristic Defect Frequency)：

外環 (BPFO)：$f_{\text{BPFO}} = \frac{n}{2}f_r\left(1-\frac{d}{D}\cos\alpha\right)$
內環 (BPFI)：$f_{\text{BPFI}} = \frac{n}{2}f_r\left(1+\frac{d}{D}\cos\alpha\right)$
滾動體 (BSF)：$f_{\text{BSF}} = \frac{D}{2d}f_r\left[1-\left(\frac{d}{D}\cos\alpha\right)^2\right]$

$n$：滾動體數量；$f_r$：轉軸頻率；$d$：滾動體直徑；$D$：節圓直徑；$\alpha$：接觸角

分析流程：振動訊號 → 帶通濾波（聚焦共振頻帶）→ 包絡偵測（Hilbert 轉換取包絡）→ FFT 分析包絡 → 在包絡譜中尋找 BPFO/BPFI 的峰值及其倍頻。

產業案例：一座風力發電機的主軸承出現早期外環缺陷。振動訊號的時域波形幾乎看不出異常，但包絡譜在 87.3 Hz（BPFO）及其 2x、3x 倍頻處出現清晰峰值，提前 3 個月預警了故障。

7.7 FFT 常見陷阱與注意事項

陷阱 1：忘記加窗

不加窗 = 矩形窗 → -13 dB 旁瓣 → 微弱訊號被洩漏淹沒。

陷阱 2：混淆 bin 與實際頻率

第 $k$ bin 對應 $f_k = k \cdot f_s/N$。常見錯誤：忘記除以 N。

陷阱 3：單邊/雙邊頻譜處理錯誤

取單邊時，除 DC 和 Nyquist 外，幅度需乘 2。

陷阱 4：以為零填充提高真正解析度

只是頻域內插。真正解析度取決於觀測時間 $\Delta f = 1/T$。

陷阱 5：取樣率不足

未加抗混疊濾波器就取樣 → 無法修正的混疊。

import numpy as np from scipy.signal import welch import matplotlib.pyplot as plt # THD 計算 fs = 10000; t = np.arange(0, 0.1, 1/fs); f0 = 50 x = np.sin(2*np.pi*f0*t) # 基頻 x += 0.08*np.sin(2*np.pi*3*f0*t) # 3次諧波 8% x += 0.05*np.sin(2*np.pi*5*f0*t) # 5次諧波 5% X = np.fft.fft(x); mag = np.abs(X[:len(X)//2])*2/len(x) bins = np.fft.fftfreq(len(x), 1/fs)[:len(x)//2] # 找各諧波峰值 V = []; for h in [1,2,3,4,5,6,7]: idx = np.argmin(np.abs(bins - h*f0)) V.append(mag[idx]) thd = np.sqrt(sum(v**2 for v in V[1:])) / V[0] * 100 print(f'THD = {thd:.2f}%') # Welch PSD f_psd, Pxx = welch(x, fs=fs, nperseg=512) plt.semilogy(f_psd, Pxx); plt.xlabel('Hz'); plt.ylabel('PSD') plt.title(f'THD = {thd:.1f}%'); plt.grid(); plt.show()
