1.1 Hilbert 空間 (Hilbert Space) 與 $L^2$ 理論

傅立葉分析的數學框架 — 為什麼正弦波能構成「基底」？

Learning Objectives

  • 定義內積空間、範數、完備性，理解 Hilbert 空間的公理化結構
  • 認識 $L^2[0,T]$ 作為傅立葉分析的自然函數空間
  • 證明複指數 $\{e^{jn\omega_0 t}\}$ 構成 $L^2$ 中的正交歸一基底 (Orthonormal Basis)
  • 從內積推導 Parseval 等式，建立「能量守恆」的嚴格基礎

內積空間 (Inner Product Space)

設 $V$ 為複數向量空間。內積 (Inner Product) $\langle \cdot, \cdot \rangle: V \times V \to \mathbb{C}$ 滿足：

$$\begin{aligned} &\text{(i) 共軛對稱：} & \langle f, g \rangle &= \overline{\langle g, f \rangle} \\ &\text{(ii) 線性（第一元）：} & \langle \alpha f + \beta g, h \rangle &= \alpha\langle f,h\rangle + \beta\langle g,h\rangle \\ &\text{(iii) 正定：} & \langle f, f \rangle &\geq 0, \;\text{且} = 0 \iff f = 0 \end{aligned}$$

由內積誘導的範數 (Norm)：$\|f\| = \sqrt{\langle f,f\rangle}$。

若內積空間在此範數下完備 (Complete)（每個 Cauchy 序列都收斂到空間內的元素），則稱為 Hilbert 空間。

$L^2[0,T]$：平方可積函數空間

傅立葉分析的自然棲息地是 $L^2$ 空間：

$$L^2[0,T] = \left\{ f: [0,T] \to \mathbb{C} \;\Big|\; \int_0^T |f(t)|^2\,dt < \infty \right\}$$

內積定義為：$\displaystyle\langle f, g \rangle = \frac{1}{T}\int_0^T f(t)\,\overline{g(t)}\,dt$

$L^2$ 的物理意義：有限能量的訊號。$\|f\|^2 = \langle f,f\rangle = \frac{1}{T}\int_0^T |f(t)|^2\,dt$ 就是平均功率。

直覺：$L^2$ 就像一個無限維的「向量空間」。每個訊號是這個空間中的一個「向量」，內積衡量兩個訊號的「相似度」，範數衡量訊號的「大小」（能量）。傅立葉分析就是在這個空間中做正交投影。

正交歸一基底：$\{e^{jn\omega_0 t}\}_{n\in\mathbb{Z}}$

令 $\phi_n(t) = e^{jn\omega_0 t}$，$\omega_0 = 2\pi/T$。關鍵定理：這組函數構成 $L^2[0,T]$ 的正交歸一基底。

展開正交性證明

計算 $\langle \phi_n, \phi_m \rangle$：

$$\langle \phi_n, \phi_m \rangle = \frac{1}{T}\int_0^T e^{jn\omega_0 t}\,\overline{e^{jm\omega_0 t}}\,dt = \frac{1}{T}\int_0^T e^{j(n-m)\omega_0 t}\,dt$$

情況一：$n = m$

$$\frac{1}{T}\int_0^T 1\,dt = 1$$

情況二：$n \neq m$

$$\frac{1}{T}\left[\frac{e^{j(n-m)\omega_0 t}}{j(n-m)\omega_0}\right]_0^T = \frac{1}{T}\cdot\frac{e^{j(n-m)2\pi} - 1}{j(n-m)\omega_0} = \frac{1-1}{j(n-m)\omega_0} = 0$$

因為 $e^{j(n-m)2\pi} = 1$（整數圈）。

結論：$\langle \phi_n, \phi_m \rangle = \delta_{nm}$（Kronecker delta），正交歸一。$\;\blacksquare$

因此，任何 $f \in L^2[0,T]$ 可以展開為：

$$f(t) = \sum_{n=-\infty}^{\infty} c_n\,\phi_n(t) = \sum_{n=-\infty}^{\infty} c_n\,e^{jn\omega_0 t}$$ $$c_n = \langle f, \phi_n \rangle = \frac{1}{T}\int_0^T f(t)\,e^{-jn\omega_0 t}\,dt$$

$c_n$ 就是 $f$ 在基底 $\phi_n$ 上的正交投影係數——完全類比有限維向量的坐標。

Parseval 等式與 Bessel 不等式

Parseval 等式 (Parseval's Identity)

$$\|f\|^2 = \frac{1}{T}\int_0^T |f(t)|^2\,dt = \sum_{n=-\infty}^{\infty} |c_n|^2$$
展開 Parseval 等式推導

將 $\|f\|^2 = \langle f, f \rangle$ 展開：

$$\langle f, f \rangle = \left\langle \sum_n c_n \phi_n,\; \sum_m c_m \phi_m \right\rangle = \sum_n \sum_m c_n \overline{c_m} \langle \phi_n, \phi_m \rangle$$

利用正交性 $\langle \phi_n, \phi_m \rangle = \delta_{nm}$：

$$= \sum_n \sum_m c_n \overline{c_m}\,\delta_{nm} = \sum_n c_n \overline{c_n} = \sum_n |c_n|^2 \quad\blacksquare$$

物理意義：時域計算的總能量（功率） = 頻域各成分能量之和。能量在正交分解中守恆。

Bessel 不等式：若只取有限項 $N$ 項，則 $\sum_{|n|\leq N} |c_n|^2 \leq \|f\|^2$。等號在 $N\to\infty$ 時成立（前提：$\{\phi_n\}$ 構成完備基底）。

互動：正交性驗證

計算 $\langle e^{jm\omega_0 t}, e^{jn\omega_0 t}\rangle$ 的實部。當 $m \neq n$ 時積分為零（正交）；$m = n$ 時為 1。

References: [1] Kreyszig, Introductory Functional Analysis with Applications, Ch.3. [2] Rudin, Real and Complex Analysis, Ch.4. [3] Oppenheim & Willsky, Signals and Systems, Ch.3.

1.2 廣義函數與分佈理論 (Distribution Theory)

為什麼 $\sin(\omega_0 t)$ 的傅立葉轉換「存在」？

Learning Objectives

  • 理解經典 FT 對 $\sin$、常數等訊號失效的原因
  • 認識測試函數空間 $\mathcal{D}$、$\mathcal{S}$ 與 tempered distributions $\mathcal{S}'$
  • 嚴格定義 Dirac delta 並推導其 FT
  • 推導 $\mathcal{F}\{1\} = 2\pi\delta(\omega)$ 和 $\mathcal{F}\{\cos\omega_0 t\}$

動機：經典 FT 的侷限

CTFT 要求 $\int_{-\infty}^{\infty}|x(t)|\,dt < \infty$（絕對可積）。但工程中最基本的訊號違反此條件：

訊號$\int|x(t)|\,dt$經典 FT？
$x(t) = A$（常數）$\infty$不存在
$x(t) = \cos(\omega_0 t)$$\infty$不存在
$x(t) = u(t)$（單位步階）$\infty$不存在

分佈理論提供了解決方案：將「函數」的概念推廣為「泛函」。

Schwartz 空間與 Tempered Distributions

Schwartz 空間 $\mathcal{S}$（快速遞減函數空間）：$\varphi \in \mathcal{S}$ 若 $\varphi$ 無窮次可微，且 $\varphi$ 及其所有導數在 $|t|\to\infty$ 時比任何多項式的倒數衰減更快：

$$\sup_{t\in\mathbb{R}} |t^m \varphi^{(n)}(t)| < \infty, \quad \forall\, m, n \geq 0$$

Tempered Distribution $T \in \mathcal{S}'$：$\mathcal{S}$ 上的連續線性泛函。記作 $T[\varphi]$ 或 $\langle T, \varphi \rangle$。

Dirac Delta 定義為泛函：

$$\delta[\varphi] = \varphi(0), \quad \forall\, \varphi \in \mathcal{S}$$

$\delta$ 不是「函數」——沒有任何經典函數 $f(t)$ 滿足 $\int f(t)\varphi(t)\,dt = \varphi(0)$ 且 $f(t) = 0$ for $t \neq 0$。它是一個純粹的泛函。

分佈的傅立葉轉換

對 $T \in \mathcal{S}'$，定義其 FT $\hat{T} \in \mathcal{S}'$ 為：

$$\hat{T}[\varphi] = T[\hat{\varphi}], \quad \forall\, \varphi \in \mathcal{S}$$
推導 $\mathcal{F}\{\delta(t)\} = 1$

設 $\hat{\varphi}(\omega) = \int_{-\infty}^{\infty}\varphi(t)\,e^{-j\omega t}\,dt$ 為 $\varphi$ 的經典 FT。

$\hat{\delta}[\varphi] = \delta[\hat{\varphi}] = \hat{\varphi}(0) = \int_{-\infty}^{\infty}\varphi(t)\,e^{0}\,dt = \int_{-\infty}^{\infty}\varphi(t)\,dt$

而常數函數 $1$ 作為 distribution 的作用：$1[\varphi] = \int 1 \cdot \varphi(t)\,dt$

因此 $\hat{\delta} = 1$，即 $\mathcal{F}\{\delta(t)\} = 1$。$\;\blacksquare$

推導 $\mathcal{F}\{1\} = 2\pi\delta(\omega)$

利用對偶性（Duality）。若 $\mathcal{F}\{f(t)\} = F(\omega)$，則 $\mathcal{F}\{F(t)\} = 2\pi f(-\omega)$。

已知 $\mathcal{F}\{\delta(t)\} = 1$，故 $\mathcal{F}\{1\} = 2\pi\delta(-\omega) = 2\pi\delta(\omega)$。$\;\blacksquare$

同理可得 $\mathcal{F}\{e^{j\omega_0 t}\} = 2\pi\delta(\omega - \omega_0)$，以及：

$$\mathcal{F}\{\cos(\omega_0 t)\} = \pi[\delta(\omega-\omega_0) + \delta(\omega+\omega_0)]$$

工程意義：$\cos(\omega_0 t)$ 的 FT 是兩個 delta 峰值，精確位於 $\pm\omega_0$。這完美符合工程直覺——純正弦波的頻譜是一條譜線。分佈理論讓這個直覺有了嚴格的數學基礎。

References: [1] Strichartz, A Guide to Distribution Theory and Fourier Transforms. [2] Folland, Real Analysis, Ch.8-9.

1.3 收斂理論 (Convergence Theory)

傅立葉級數「收斂」到底是什麼意思？

Learning Objectives

  • 區分逐點、均勻、$L^2$、幾乎處處四種收斂模式
  • 陳述 Dirichlet 條件與 Carleson 定理
  • 量化收斂速率與訊號平滑度的關係
  • 嚴格分析 Gibbs 現象的 8.95% 過衝

四種收斂模式

令 $S_N(t) = \sum_{|n|\leq N} c_n e^{jn\omega_0 t}$ 為部分和。收斂到 $f(t)$ 的含義取決於收斂模式：

模式定義條件工程意義
逐點 (Pointwise)$\forall t$: $S_N(t) \to f(t)$Dirichlet 條件每個時間點都收斂
均勻 (Uniform)$\sup_t |S_N(t)-f(t)|\to 0$$f$ 連續 + 絕對收斂最強；方波做不到（Gibbs）
$L^2$ (Mean-Square)$\int|S_N-f|^2 dt \to 0$$f \in L^2$（總是成立）能量意義收斂；最常用
幾乎處處 (a.e.)$S_N(t)\to f(t)$ except 零測集$f\in L^2$（Carleson 1966）除可忽略的點外都收斂

收斂速率與平滑度

關鍵定理：訊號越平滑，傅立葉係數衰減越快。

$$\text{若 } f \in C^k \text{（$k$ 次連續可微）} \implies |c_n| = O(|n|^{-(k+1)})$$
訊號連續性$c_n$ 衰減收斂速度
方波不連續$O(1/n)$慢（Gibbs）
三角波$C^0$（連續但不可微）$O(1/n^2)$中等
拋物線波$C^1$$O(1/n^3)$快
$C^\infty$ 函數無窮次可微超代數衰減極快

工程啟示：這解釋了為什麼實際系統中方波需要非常大的頻寬（諧波衰減慢），而平滑的 raised-cosine 脈衝只需要有限頻寬（諧波衰減快）。

Gibbs 現象的嚴格分析

在方波不連續點附近，$S_N$ 的最大過衝趨向：

$$\text{overshoot} = \frac{1}{\pi}\int_0^\pi \frac{\sin u}{u}\,du - \frac{1}{2} \approx 0.0895 \approx 8.95\%$$
推導 Gibbs 過衝比例

方波 $f(t) = \text{sgn}(\sin t)$ 的部分和在 $t = \pi/(N+1)$ 處取極大值。可以證明：

$$S_N\!\left(\frac{\pi}{N+1}\right) = \frac{2}{\pi}\sum_{k=0}^{N}\frac{\sin\left(\frac{(2k+1)\pi}{2(N+1)}\right)}{2k+1} \cdot \frac{2}{2(N+1)}$$

當 $N\to\infty$，此 Riemann 和趨近於：

$$\frac{2}{\pi}\int_0^{\pi}\frac{\sin u}{u}\,du = \frac{2}{\pi}\cdot\text{Si}(\pi) \approx \frac{2}{\pi}(1.8519) \approx 1.1790$$

而理想方波值為 1，故過衝 $\approx 17.90\%$ of half-amplitude = $8.95\%$ of full amplitude。$\;\blacksquare$

互動：收斂速率比較

比較方波 ($1/n$)、三角波 ($1/n^2$)、拋物線波 ($1/n^3$) 的傅立葉級數收斂速度。

References: [1] Carleson, On convergence and growth of partial sums of Fourier series, Acta Math., 1966. [2] Körner, Fourier Analysis, Cambridge.

1.4 不確定性原理 (Uncertainty Principle)

Heisenberg-Gabor 不等式的嚴格證明

Learning Objectives

  • 定義時域展度 $\Delta t$ 與頻域展度 $\Delta\omega$ 的嚴格數學意義
  • 從 Cauchy-Schwarz 不等式完整推導 $\Delta t\cdot\Delta\omega \geq \frac{1}{2}$
  • 證明 Gaussian 達到等號（最小不確定性）
  • 連結到 STFT 解析度限制與雷達模糊函數

嚴格陳述

定義時域與頻域的「展度」為均方根寬度（二階矩）：

$$\Delta t^2 = \frac{\int_{-\infty}^{\infty} t^2|f(t)|^2\,dt}{\int_{-\infty}^{\infty}|f(t)|^2\,dt}, \quad \Delta\omega^2 = \frac{\int_{-\infty}^{\infty} \omega^2|F(\omega)|^2\,d\omega}{\int_{-\infty}^{\infty}|F(\omega)|^2\,d\omega}$$

Heisenberg-Gabor 不等式

$$\boxed{\Delta t \cdot \Delta\omega \geq \frac{1}{2}}$$

等號成立若且唯若 $f(t) = Ce^{-\alpha t^2}$（Gaussian）

完整證明

展開完整推導（使用 Cauchy-Schwarz）

Step 1：不失一般性，假設 $\|f\| = 1$（歸一化），且 $f$ 的質心在原點（否則平移）。

Step 2：利用微分性質 $\mathcal{F}\{tf(t)\} = j\frac{d}{d\omega}F(\omega)$，以及 $\mathcal{F}\{f'(t)\} = j\omega F(\omega)$。由 Parseval：

$$\Delta t^2 = \int t^2|f(t)|^2\,dt, \quad \Delta\omega^2 = \int \omega^2|F(\omega)|^2\,d\omega = \frac{1}{2\pi}\int|f'(t)|^2\,dt$$

Step 3：對 $tf(t)$ 和 $f'(t)$ 使用 Cauchy-Schwarz 不等式：

$$\left|\int tf(t)\overline{f'(t)}\,dt\right|^2 \leq \int t^2|f|^2\,dt \cdot \int |f'|^2\,dt = \Delta t^2 \cdot 2\pi\Delta\omega^2$$

Step 4：計算左邊。利用分部積分：

$$\int tf(t)\overline{f'(t)}\,dt = \int t \cdot \frac{1}{2}\frac{d}{dt}|f(t)|^2\,dt$$ $$= \left[\frac{t}{2}|f|^2\right]_{-\infty}^{\infty} - \frac{1}{2}\int|f(t)|^2\,dt = 0 - \frac{1}{2}\|f\|^2 = -\frac{1}{2}$$

（邊界項消失因為 $f \in L^2$ 意味著 $t|f(t)|^2 \to 0$。）

Step 5：代入得 $\frac{1}{4} \leq \Delta t^2 \cdot 2\pi\Delta\omega^2$，即 $\Delta t \cdot \Delta\omega \geq \frac{1}{2}$（若用 $\Delta f$ 代替 $\Delta\omega$，則 $\Delta t \cdot \Delta f \geq \frac{1}{4\pi}$）。$\;\blacksquare$

等號條件：Cauchy-Schwarz 等號 $\iff$ $f'(t) = -2\alpha t f(t)$ $\iff$ $f(t) = Ce^{-\alpha t^2}$。

互動：Gaussian 的 $\Delta t \cdot \Delta f$ 乘積

調整 Gaussian $e^{-\alpha t^2}$ 的 $\alpha$ 參數。觀察：時域越窄 → 頻域越寬，但乘積恆為 $\frac{1}{4\pi}$。

References: [1] Folland & Sitaram, The uncertainty principle: a mathematical survey, J. Fourier Anal. Appl., 1997. [2] Gabor, Theory of Communication, J. IEE, 1946.

2.1 傅立葉級數 (Fourier Series)

週期訊號的頻域表示 — 完整理論

Learning Objectives

  • 推導三角形式與複數形式的係數公式
  • 利用對稱性（奇/偶/半波）簡化係數計算
  • 計算方波、三角波、鋸齒波的傅立葉係數
  • 陳述微分與積分對傅立葉級數的影響

兩種形式

三角形式：

$$f(t) = \frac{a_0}{2} + \sum_{n=1}^{\infty}\left[a_n\cos(n\omega_0 t) + b_n\sin(n\omega_0 t)\right]$$

複指數形式：

$$f(t) = \sum_{n=-\infty}^{\infty}c_n\,e^{jn\omega_0 t}, \quad c_n = \frac{1}{T}\int_0^T f(t)\,e^{-jn\omega_0 t}\,dt$$

關係：$c_0 = a_0/2$，$c_n = (a_n - jb_n)/2$，$c_{-n} = \overline{c_n}$（實數訊號）

對稱性簡化

訊號對稱性結果級數只含
偶函數 $f(-t)=f(t)$$b_n=0$只有 cosine
奇函數 $f(-t)=-f(t)$$a_n=0$只有 sine
半波對稱 $f(t+T/2)=-f(t)$偶數次諧波為零只有奇數次

經典波形係數推導

方波 (Square Wave) 係數推導

週期 $T$、振幅 $\pm 1$ 的方波：$f(t) = 1$ for $0 < t < T/2$，$f(t) = -1$ for $T/2 < t < T$。

$$c_n = \frac{1}{T}\left[\int_0^{T/2}e^{-jn\omega_0 t}\,dt - \int_{T/2}^{T}e^{-jn\omega_0 t}\,dt\right]$$ $$= \frac{1}{T}\cdot\frac{1}{-jn\omega_0}\left[(e^{-jn\pi}-1) - (e^{-jn2\pi}-e^{-jn\pi})\right]$$ $$= \frac{1}{-jn\omega_0 T}\left[2e^{-jn\pi}-1-1\right] = \frac{2((-1)^n-1)}{-jn\cdot 2\pi}$$

$n$ 偶數 → $c_n = 0$。$n$ 奇數 → $c_n = \frac{-4}{-jn\cdot 2\pi} = \frac{2}{jn\pi}$。

轉回 sine 形式：$b_n = -2j\,c_n = 4/(n\pi)$（奇數 $n$），$a_n = 0$。

$$\boxed{f(t) = \frac{4}{\pi}\sum_{n=1,3,5,\ldots}\frac{1}{n}\sin(n\omega_0 t)} \quad\blacksquare$$

微分與積分

$$f'(t) = \sum_{n} jn\omega_0\, c_n\, e^{jn\omega_0 t} \quad\Longrightarrow\quad \text{微分在頻域 = 乘以 } jn\omega_0$$

這意味著不連續的函數（如方波）有 $c_n \sim 1/n$（慢衰減），而可微函數有 $c_n \sim 1/n^2$ 或更快。每多一階可微，係數多衰減一個 $1/n$ 因子。

2.2 連續時間傅立葉轉換 (CTFT)

從傅立葉級數到非週期訊號 — 令 $T\to\infty$

Learning Objectives

  • 從 FS 推導 CTFT（$T\to\infty$ 極限過程）
  • 證明時移、卷積、Parseval 等核心性質
  • 推導 rect↔sinc 和 Gaussian↔Gaussian 轉換對

從 FS 到 CTFT 的推導

展開完整推導

週期訊號 $f_T(t) = \sum_n c_n e^{jn\omega_0 t}$，$c_n = \frac{1}{T}\int_{-T/2}^{T/2}f(t)e^{-jn\omega_0 t}dt$。

定義 $F_T(\omega) = Tc_n|_{\omega=n\omega_0} = \int_{-T/2}^{T/2}f(t)e^{-j\omega t}dt$。

則 $f_T(t) = \frac{1}{T}\sum_n F_T(n\omega_0)e^{jn\omega_0 t} = \frac{1}{2\pi}\sum_n F_T(n\omega_0)\,\omega_0\, e^{jn\omega_0 t}$。

令 $T\to\infty$：$\omega_0 = 2\pi/T \to d\omega$，離散求和 → 積分：

$$f(t) = \frac{1}{2\pi}\int_{-\infty}^{\infty}F(\omega)\,e^{j\omega t}\,d\omega, \quad F(\omega) = \int_{-\infty}^{\infty}f(t)\,e^{-j\omega t}\,dt \quad\blacksquare$$

2.3 離散時間傅立葉轉換 (DTFT)

連接連續世界與離散世界的橋樑

Learning Objectives

  • 定義 DTFT 並理解其頻域的 $2\pi$ 週期性
  • 從 CTFT + 取樣推導 DTFT（頻域週期化）
  • 區分 DTFT（連續頻率）與 DFT（離散頻率取樣）

定義

$$X(e^{j\omega}) = \sum_{n=-\infty}^{\infty}x[n]\,e^{-j\omega n}$$

$X(e^{j\omega})$ 是 $\omega$ 的連續函數，以 $2\pi$ 為週期。

與 CTFT 的關係：若 $x[n] = x_c(nT_s)$（取樣），則：

$$X(e^{j\omega}) = \frac{1}{T_s}\sum_{k=-\infty}^{\infty}X_c\!\left(\frac{\omega - 2\pi k}{T_s}\right)$$

取樣造成頻譜的週期化。若 $X_c$ 的頻寬超過 $\pi/T_s$（Nyquist 頻率），相鄰副本重疊 → 混疊。

DTFT vs DFT

DFT 是 DTFT 在頻率軸上的均勻取樣：

$$X[k] = X(e^{j\omega})\big|_{\omega = 2\pi k/N} = \sum_{n=0}^{N-1}x[n]\,e^{-j2\pi kn/N}$$

DTFT 給出完整的連續頻譜；DFT 只是從中取 N 個等距樣本。零填充增加 DFT 的取樣密度（看到更多 DTFT 的細節），但不改變 DTFT 本身。

互動：DTFT vs DFT

藍色實線 = DTFT（高密度 DFT 近似），紅點 = N 點 DFT 的取樣。增加 N 點看到更多 DTFT 細節。

2.4 DFT 與 FFT

離散傅立葉轉換與快速演算法

Learning Objectives

  • DFT 矩陣觀點：$\mathbf{X} = \mathbf{W}_N\mathbf{x}$，酉性質
  • 圓卷積 vs 線性卷積
  • Cooley-Tukey radix-2 分治推導

DFT 矩陣

$$X[k] = \sum_{n=0}^{N-1}x[n]\,W_N^{kn}, \quad W_N = e^{-j2\pi/N}$$ $$\mathbf{X} = \mathbf{W}_N\mathbf{x}, \quad [\mathbf{W}_N]_{kn} = W_N^{kn}$$

$\frac{1}{\sqrt{N}}\mathbf{W}_N$ 為酉矩陣（Unitary）：$\mathbf{W}_N^H\mathbf{W}_N = N\mathbf{I}$

圓卷積 (Circular Convolution)

$$\text{DFT}\{x \circledast y\} = X[k]\cdot Y[k]$$

DFT 對應的是圓卷積，不是線性卷積。要用 DFT 計算線性卷積，必須零填充到 $\geq N_x + N_y - 1$。

Radix-2 FFT 實作

// Radix-2 Cooley-Tukey FFT（本站核心引擎） function fft(re_in, im_in) { const N = re_in.length; if (N <= 1) return { re: [...re_in], im: [...im_in] }; const re = new Float64Array(N), im = new Float64Array(N); const bits = Math.round(Math.log2(N)); for (let i = 0; i < N; i++) { // 位元反轉 let r = 0; for (let b = 0; b < bits; b++) r = (r << 1) | ((i >> b) & 1); re[r] = re_in[i]; im[r] = im_in[i]; } for (let s = 2; s <= N; s *= 2) { // 蝴蝶運算 const h = s/2, a = -2*Math.PI/s; for (let i = 0; i < N; i += s) for (let j = 0; j < h; j++) { const wr=Math.cos(a*j), wi=Math.sin(a*j); const tr=wr*re[i+j+h]-wi*im[i+j+h]; const ti=wr*im[i+j+h]+wi*re[i+j+h]; re[i+j+h]=re[i+j]-tr; im[i+j+h]=im[i+j]-ti; re[i+j]+=tr; im[i+j]+=ti; } } return { re: Array.from(re), im: Array.from(im) }; }

2.5 Z 轉換 (Z-Transform)

離散時間系統的統一分析框架

Learning Objectives

  • 定義 Z 轉換及收斂域 (ROC)
  • 理解 DFT = Z 轉換在單位圓上的取樣
  • 極零點分析與穩定性條件

定義

$$X(z) = \sum_{n=-\infty}^{\infty}x[n]\,z^{-n}, \quad z \in \mathbb{C}$$

DFT 是 $z = e^{j2\pi k/N}$ 的特例（單位圓上等距取樣）

ROC（收斂域）：使級數絕對收斂的 $z$ 值集合。ROC 的形狀決定了序列的性質：

  • 因果序列：ROC 是某個圓的外部（$|z| > r$）
  • 穩定系統：ROC 必須包含單位圓 → 所有極點在單位圓內

互動：極零點圖與頻率響應

調整極點位置，觀察頻率響應（= Z 轉換在單位圓上的值）如何變化。

2.6 取樣定理 (Sampling Theorem)

從 Poisson 求和公式嚴格推導 Shannon 定理

Learning Objectives

  • 推導取樣 = 頻域週期化（Poisson 求和公式）
  • 從週期化推導 Nyquist 條件
  • 推導 sinc 重建公式

嚴格推導

從 Poisson 求和到 Shannon 定理

Step 1：取樣 = 乘以脈衝串

$$x_s(t) = x(t)\cdot\sum_{n=-\infty}^{\infty}\delta(t - nT_s) = \sum_n x(nT_s)\,\delta(t-nT_s)$$

Step 2：頻域（利用乘法→卷積，$\mathcal{F}\{\text{impulse train}\} = \frac{2\pi}{T_s}\text{impulse train}$）

$$X_s(\omega) = \frac{1}{T_s}\sum_{k=-\infty}^{\infty}X(\omega - k\omega_s), \quad \omega_s = 2\pi/T_s$$

取樣造成頻譜的週期性重複，間距 $\omega_s$。

Step 3：Nyquist 條件

若 $X(\omega) = 0$ for $|\omega| > \omega_{\max}$（帶限訊號），且 $\omega_s > 2\omega_{\max}$，則相鄰副本不重疊，可用理想低通濾波器恢復：

$$\boxed{f_s \geq 2f_{\max}}$$

Step 4：sinc 重建

$$x(t) = \sum_{n=-\infty}^{\infty}x(nT_s)\,\text{sinc}\!\left(\frac{t-nT_s}{T_s}\right) \quad\blacksquare$$

互動：Aliasing

2.7 轉換關係總覽

FS ↔ CTFT ↔ DTFT ↔ DFT ↔ Z-Transform 的完整關係鏈

關係圖

轉換時域頻域連結操作
FS連續、週期離散（$c_n$）$T\to\infty$（去週期化）
CTFT連續、非週期連續
CTFT→DTFT取樣：$x[n]=x_c(nT_s)$ → 頻域週期化 $X(e^{j\omega}) = \frac{1}{T_s}\sum_k X_c(\frac{\omega-2\pi k}{T_s})$
DTFT離散、非週期連續、$2\pi$-週期截斷 $N$ 點（頻率取樣）
DFT離散、$N$-週期離散、$N$-週期
Z-Transform離散$z$ 平面函數DFT = 單位圓取樣

核心洞察：時域的取樣導致頻域的週期化；時域的截斷（有限長度）導致頻域的離散化。DFT 是兩者都做了：既取樣又截斷，所以時域和頻域都是離散且週期的。
